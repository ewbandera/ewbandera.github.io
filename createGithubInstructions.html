Github Instructions

  • Navigate to github.com
  • Click Sign Up, and create account with a username you'd like to use for the website

  • Once account is created, click "Create a new Repository"
  • Name the repository {username}.github.io where {username} is your account username you set up in step 1
    • Note that if you used any uppercase letters in your username, convert them to lower case
    • Note that this will automatically create a public website at the URL https://{username}.github.com
  • Click a link for "uploading an existing file" - this will bring up a drag and drop

  • This should include at a minimum a default document named index.html which is your homepage

  • Drag and drop desktop files to the github window

  • Click the "Commit" button within github to confirm the uploads

  • Type https://{username}.github.io into the browser to see if it is working
